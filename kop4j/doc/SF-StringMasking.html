String Masking

General description

This protection will mask string value, by replacing them by an unmasking method.
The value is recomputed at the first use of the string. The penalty on the performance is mainly at the first string usage.

When targeting static final String fields, the strings are unmasked when the class is loaded.

Security note

Security

Take care to remove or mask the strings in log messages

Security

String masking on fields limits the opacity source to the container class only.

Protection Configuration

String Masking is a global protection, then enabled by default on methods, but disabled by default on fields.
The string masking can be disabled or enabled on a sub part.

  • Command to disable : -kop-mask-string,disable <classic designator>
  • Command to enable : -kop-mask-string,enable <classic designator> or -kop-mask-string, <classic designator>

The order of commands is important when enabling/disabling string masking.
For example:

-kop-mask-string,disable class ** { *; }
-kop-mask-string, class package.name.Main { public static void main(...); }

disables string masking on all methods except package.name.Main.main, but the following

-kop-mask-string,enable class package.name.Main { public static void main(...); }
-kop-mask-string,disable class ** { *; }

disables string masking on ALL methods and fields (the second line would override the first one).

The complexity of the string masking algorithm can be adjusted with the complexity option.
For example:

-kop-mask-string,enable,complexity:5 class package.name.Main { public static void main(...); }

The default complexity is set to 2.

Masked strings are grouped by 'family', i.e. strings that share some characteristics (typically: include, exclude and complexity of the methods they are referenced from).
To prevent all strings from being grouped in a single class, the maximum number of strings per group can be adjusted with:

-kop-mask-string-batch-size 100

Examples

Enable string masking on all static final String fields of the com.nagra.example.Constants interface:

-kop-mask-string,enable interface com.nagra.example.Constants { static final java.lang.String <fields>; }
