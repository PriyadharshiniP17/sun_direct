Installing KOP4J

To install KOP4J, unzip to a directory.

License management

KOP4J requires a valid license to run.

This .lic file can be located:

  • In the user's home directory (recommended)
  • In the current directory
  • In a directory listed in the LM_LICENSE_PATH environment variable.
  • In a file listed in the LM_LICENSE_FILE environment variable.

Note that:

  • The license is required to apply kop4j.
  • The license is not required to run the protected program.
    For example, programs protected by kop4j will continue to be functional past kop4j's license expiration.
