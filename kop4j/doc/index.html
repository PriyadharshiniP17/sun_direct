KOP for Java 1.12.1

What's this tool ?

KOP for Java (KOP4J) is a Java code obfuscation tool.
It makes program hard to reverse, while keeping the program semantically equivalent to the original.

How does it work ?

KOP4J works by modifying the Java program after compilation, but before packaging.

Applying KOP4J is done by integrating it in the build chain, which depends on how your application is built.
You can find more information about the integration here:

  • Applying KOP4J on Android and plain Java projects
  • Applying KOP4J on Android projects

KOP4J specific options

KOP4J understands most classic proguard options.
It also adds its own KOP4J options commands to fine-tune the protection.
